## 💡 Sobre o Projeto
* Nome: Sabor Macabro

* Descrição: Confeitaria temática inspirada no terror/Halloween que combina venda de doces criativos com ensino de receitas. O site oferece uma experiência de compra e aprendizado, incluindo versões adaptadas com menos açúcar ou ingredientes alternativos, tornando a confeitaria mais inclusiva.

* Objetivo: Desenvolver um site temático moderno e responsivo que una entretenimento, gastronomia e acessibilidade, proporcionando uma experiência imersiva para usuários interessados em confeitaria criativa e estética Halloween. O projeto também teve como objetivo aplicar conceitos de UX/UI, acessibilidade, responsividade e desenvolvimento front-end utilizando ferramentas de inteligência artificial no processo de design e programação.

* Tecnologias utilizadas:

- Base 44 AI
- Claude AI
- GitHub
- Vercel

## ⚙ Prompt 
CONTEXTO: Desenvolver um site chamado “Sabor Macabro”, uma confeitaria temática inspirada em terror, Halloween e estética gótica suave. O projeto combina venda de doces criativos com ensino de receitas, oferecendo também versões adaptadas para pessoas com restrição de açúcar, incluindo opções para diabéticos e alternativas com ingredientes diferenciados. OBJETIVO: Criar uma plataforma moderna, responsiva e acessível que una experiência de compra, aprendizado culinário e identidade visual temática. O site deve transmitir criatividade, diversão e inclusão alimentar. PÚBLICO-ALVO: Jovens e adultos que gostam de Halloween, terror leve e estética alternativa, pessoas interessadas em confeitaria criativa, usuários que buscam receitas adaptadas com menos açúcar ou ingredientes alternativos e consumidores de doces artesanais temáticos. REQUISITOS VISUAIS: Estilo Halloween cute/gótico moderno, paleta de cores roxo, preto, laranja, vinho e branco, elementos decorativos inspirados em terror leve como morcegos, abóboras, fantasmas, velas, luas e doces temáticos, visual moderno e organizado, cards estilizados para produtos e receitas, efeitos suaves de animação e hover, fonte decorativa apenas em títulos e fonte legível no restante do conteúdo. REQUISITOS FUNCIONAIS: Página inicial, catálogo de doces, área de receitas, sessão de receitas adaptadas para diabéticos, página “Sobre”, página de contato, barra de navegação responsiva, sistema de categorias para receitas, botões de chamada para ação e footer com redes sociais e informações. REQUISITOS TÉCNICOS: Utilizar HTML, CSS e JavaScript, estrutura organizada de pastas, página principal definida como index.html, código responsivo para dispositivos móveis, compatibilidade com navegadores modernos e possibilidade de expansão futura para PWA. NAVEGAÇÃO: Navegação simples e intuitiva, menu visível em desktop e mobile, rolagem fluida entre seções e hierarquia clara de informações. ACESSIBILIDADE: Contraste adequado entre texto e fundo, compatibilidade para usuários com daltonismo, textos alternativos em imagens, boa legibilidade, navegação acessível por teclado e botões e links bem identificados. RESULTADO ESPERADO: Gerar um site moderno, funcional e visualmente marcante para a confeitaria Sabor Macabro, oferecendo experiência temática imersiva, acessibilidade e conteúdo educativo relacionado à confeitaria criativa e receitas adaptadas.
